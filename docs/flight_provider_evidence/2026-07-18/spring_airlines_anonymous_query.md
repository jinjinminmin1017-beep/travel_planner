# 春秋航空匿名公开查询验证

验证时间：2026-07-18（Asia/Shanghai）

## 结论

- 春秋航空公开订票页无需登录即可查询航班和票价。
- 官网前端使用 `POST https://flights.ch.com/Flights/SearchByTime`，表单只包含路线、日期、人数和公开搜索开关；成功请求不需要账号、Cookie、token、签名、设备指纹或验证码材料。
- 项目已以独立 Provider `airline_9c_public_query` 接入该匿名公开查询，许可状态为 `APPROVED`，商业使用保持 `false`，请求频率限制为 1 QPS。
- 若官网返回验证码、人机挑战、HTTP 429 或超时，Provider 会明确失败，不尝试绕过。

## 后台接口验证

- 请求：上海 `SHA` → 广州 `CAN`，出发日期 2026-07-23，成人 1 人，CNY。
- 响应：HTTP 200，业务码 `Code=0`，`Route` 含 6 组航班。
- 航班事实字段：`No`、`DepartureTime`、`ArrivalTime`、`DepartureAirportCode`、`ArrivalAirportCode`、`FlightTimeM`。
- 舱价事实字段：`AircraftCabins[].AircraftCabinInfos[].Name/Price/Remain`。
- `Remain=0` 在官网仍对应可售展示价，因此实现将其解释为“可售但数量未知”；只有正整数才记录为有限余量，避免把未知误报为售罄。

## 项目 smoke

项目 Provider 首次以 30 秒超时运行时记录上游读取超时；将实际配置调整为 60 秒后，同一路线成功返回：

```text
source=airline_9c_public_query
route=SHA->CAN
flight=9C8835
price=CNY 390.00
cabins=经济舱 AVAILABLE/LIMITED
result=PASS
```

Provider 对候选按价格、出发时间稳定排序，因此规划器的小结果窗口仍会保留样本最低价 9C8931（¥370）和较早航班。

## 数据与安全边界

- 只查询航班、公开票价和舱位展示信息，不登录、不下单、不支付、不处理乘客实名信息。
- 原始响应入库前执行敏感字段递归脱敏，请求键只保存 SHA-256 指纹。
- ENV 只保存非敏感运行配置；请求结构和解析规则由代码实现，不能靠修改 `status` 或伪造技术门禁启用未实现航司。
- 本记录不包含完整响应、Cookie、token、指纹或动态风控材料。
